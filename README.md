#Akan Name Generator

This application enables users to know their Ghanaian name based on their gender and the period they were born.

## Project Description
In Ghanaian culture, children were named after the day of the week they were born.This project uses a mathematical formula to calculate the day of the week based of the birthdate of the user and the year they were born depending on the user's gender as well.

## Built With:
* HTML
* CSS
* JAVASCRIPT
